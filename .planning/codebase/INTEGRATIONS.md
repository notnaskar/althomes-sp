# External Integrations

**Last mapped:** 2026-04-24

---

## Sanity CMS (Primary Data Store)

**Type:** Headless CMS / Content Lake
**Package:** `next-sanity` ^12.3.0, `sanity` ^5.21.0

- **Project ID:** `NEXT_PUBLIC_SANITY_PROJECT_ID` (env)
- **Dataset:** `NEXT_PUBLIC_SANITY_DATASET` (env, default `production`)
- **API Version:** `2025-10-23` (live content), standard version in `src/sanity/env.ts`
- **CDN:** `cdn.sanity.io` — enabled via `useCdn: true` in `src/sanity/lib/client.ts`
- **Image CDN:** `cdn.sanity.io` — whitelisted in `next.config.ts` `remotePatterns`
- **Client:** `src/sanity/lib/client.ts` — `createClient` from `next-sanity`
- **Live Content API:** `src/sanity/lib/live.ts` — `defineLive` with `serverToken` + `browserToken`
- **Query language:** GROQ via `groq` tagged template literal
- **Stega encoding:** enabled, studio URL `/admin`

### Draft Mode
- **Enable:** `GET /api/draft-mode/enable` — `src/app/(frontend)/api/draft-mode/enable/route.ts`
- **Disable:** `GET /api/draft-mode/disable` — `src/app/(frontend)/api/draft-mode/disable/route.ts`
- **Token:** `SANITY_API_READ_TOKEN` (env) — "Viewer" permissions
- **Preview perspective:** switches to `drafts` when `draftMode().isEnabled` or in `dev`

### Sanity Studio (Embedded)
- **Route:** `/admin/[[...tool]]` — `src/app/(studio)/admin/[[...tool]]/page.tsx`
- **Config:** `sanity.config.ts` at root
- **Plugins:** `@sanity/assist`, `@sanity/dashboard`, `@sanity/vision`, `@sanity/code-input`, `sanity-plugin-dashboard-widget-vercel`

## Vercel (Deployment Platform)

**Type:** Hosting / Serverless
**Detection:** `process.env.VERCEL_ENV` used in `src/lib/env.ts` to detect preview mode
**Dashboard widget:** `sanity-plugin-dashboard-widget-vercel` — Vercel deploy status in Sanity Studio
**OG Image:** generated via `/api/og` route (`src/app/(frontend)/api/og/route.tsx`)

## Form Endpoint (CMS-Configured)

**Type:** External form submission
**Location:** `src/ui/modules/form-module/contact.tsx`
**Mechanism:** HTML `<form action={form.endpoint} method="POST">` — endpoint URL stored in Sanity `form` document
**No server-side proxy** — POSTs directly to whatever URL is set in CMS
